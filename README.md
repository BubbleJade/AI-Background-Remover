# 🪄 AI 智能一键抠图工具 (AI Background Remover)

> **简介**：这是一个基于深度学习模型（U²-Net）的轻量级桌面应用，旨在为用户提供免费、离线、高精度的图像背景去除服务。无需上传图片到云端，保护隐私，一键生成透明背景 PNG。

### 🌟 核心功能 (Features)

  * **⚡️ 一键自动抠图**：利用工业级开源库 `rembg`，毫秒级识别前景。
  * **🖼️ 高清导出**：支持保留原始分辨率，导出为带透明通道的 PNG 格式。
  * **👁️ 可视化蒙版**：提供“原图/结果/蒙版”三种视图切换，方便检查边缘识别准确度。
  * **🖱️ 极简 GUI**：基于 Tkinter 原生界面，无广告，无复杂设置，即开即用。
  * **🔒 隐私安全**：所有计算在本地运行，图片数据不经过任何服务器。

-----

### 🛠️ 技术栈 (Tech Stack)

  * **语言**：Python 3.8+
  * **GUI 框架**：Tkinter (Python 内置)
  * **核心算法**：[rembg](https://github.com/danielgatis/rembg) (基于 U²-Net 架构)
  * **图像处理**：Pillow (PIL), NumPy

### 🚀 快速开始 (Quick Start)

#### 1\. 环境准备

确保您的电脑已安装 Python 3.8 或更高版本。

#### 2\. 安装依赖

在项目根目录下打开终端（Terminal 或 CMD），运行：

```bash
pip install -r requirements.txt
```

*(注：安装 `rembg` 时可能会自动安装 PyTorch 等依赖，需保持网络通畅)*

#### 3\. 运行软件

```bash
python main.py
```

### 📖 用户使用指南 (User Guide)

#### 第一步：启动软件

运行 `main.py` 后，您将看到如下界面。界面分为顶部操作栏和中间预览区。

#### 第二步：加载图片

1.  点击左上角的 **"1. 选择图片"** 按钮。
2.  在弹出的文件框中选择一张 JPG 或 PNG 图片（建议选择主体清晰的照片）。
3.  **注意**：左侧面板将立即显示您选择的原图。

#### 第三步：等待自动处理

  * 软件会自动开始抠图（此时状态栏显示“正在处理中...”）。
  * **⚠️ 首次运行特别说明**：
    第一次使用时，程序会自动从 GitHub 下载预训练模型（`u2net.onnx`，约 170MB）。
      * 这可能需要 **1-5 分钟**，具体取决于您的网速。
      * **请勿关闭程序**，可以在终端窗口查看下载进度。
      * 模型下载只需一次，后续使用将是秒级响应。

#### 第四步：预览与检查

处理完成后，右侧面板将显示去背后的结果（默认显示棋盘格背景以展示透明效果）。

  * **切换视图**：在底部单选框中，您可以点击 **"查看可视化蒙版"**。
      * **白色区域** = 保留部分
      * **黑色区域** = 移除部分

#### 第五步：保存结果

1.  确认效果满意后，点击右上角变绿的 **"2. 导出 PNG"** 按钮。
2.  选择保存路径，文件将保存为透明背景的 `.png` 图片。

### ❓ 常见问题 (Q\&A)

**Q: 为什么第一次点击后软件像卡死了一样？**
A: 这是因为后台正在下载 AI 模型文件。请关注您的控制台/终端输出，下载完成后界面会自动恢复响应。

**Q: 支持批量处理吗？**
A: 当前版本为轻量级 GUI 版，仅支持单张精细处理。如需批量处理，可直接修改 `core.py` 进行循环调用。

**Q: 抠图效果边缘不清晰怎么办？**
A: U²-Net 对毛发等复杂边缘处理能力极强，但如果原图主体与背景颜色过于接近，可能会有误差。建议使用对比度较高的图片。
